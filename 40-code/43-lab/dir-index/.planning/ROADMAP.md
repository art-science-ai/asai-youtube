# Roadmap: Directory Indexer

## Overview

A single-phase project to build a lightweight CLI tool that creates and maintains markdown indices for external directories. The tool will traverse target directories, generate markdown files with YAML frontmatter containing file metadata, and preserve manual notes on re-runs while updating metadata.
