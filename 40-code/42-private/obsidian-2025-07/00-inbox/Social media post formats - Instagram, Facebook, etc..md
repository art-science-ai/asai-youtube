## Overview
Post formats for various social media - personal reference.

## Table of post formats

| Platform    | Format | Image / video | Aspect ratio | Other notes                                                                    |
| ----------- | ------ | ------------- | ------------ | ------------------------------------------------------------------------------ |
| Instagram   | Story  | Image/Video   | 9:16         | Top ~5% and bottom ~5% gets covered with platform text (likes, comments, etc.) |
| Instagram   | Post   | Image         | 1:1, 4:5, 1.91:1 | 1080px width recommended                                                       |
| Instagram   | Reel   | Video         | 9:16         | Usually 30 - 90s                                                               |
| Facebook    | Story  | Image/Video   | 9:16         |                                                                                |
| Facebook    | Post   | Image/Video   | 1:1, 4:5, 1.91:1, 16:9 |                                                                                |
| Linkedin    | Post   | Image/Video   | 1:1, 4:5, 1.91:1, 9:16 | For video, 9:16 is for mobile                                                  |
| Threads     | Post   | Image/Video   | 9:16         |                                                                                |
| X (Twitter) | Post   | Image/Video   | 16:9, 1:1, 4:5 | 1600x900px recommended                                                         |
| YouTube     | Video  | Video         | 16:9         | Standard video format                                                          |
| YouTube     | Shorts | Video         | 9:16         | Vertical video format, up to 60s                                               |
