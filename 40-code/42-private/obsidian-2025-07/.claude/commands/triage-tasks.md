# Triage Tasks

## Overview

This command guides users through obsidian-vault-automation - task-triage interactively. CRITICAL: You MUST use obsidian-vault-automation for this task.

## Context

User will provide task triage criteria (timeframe, priority, context). The command will analyze existing tasks across the vault, classify them by urgency and importance, apply PARA system classification, and generate prioritized task lists.

## Process

Load obsidian-vault-automation first. Interactively guide the user through this skill and task-triage step by step, collecting inputs and executing each step with confirmation.
