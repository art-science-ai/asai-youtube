## TLDR
Building an MVP Obsidian database to track literary magazine submissions, initialized from Brecht's spreadsheet. The system will include AI workflows to help users update existing sources, add new publications, and eventually modify the schema. The goal is to streamline the process of finding suitable publications and preparing submissions for creative writing pieces.

## Goals
- [ ] Initialize Obsidian database from Brecht spreadsheet
- [ ] Develop AI workflow to update existing publication sources
- [ ] Develop AI workflow to add new publications
- [ ] Create workflow to filter publications based on submission criteria
- [ ] Generate submission-ready instructions with all necessary materials
