Features I want
- Must have features
	- Cross platform: desktop + web + mobile apps (at least iOS)
	- Export history
	- Edit previous days
	- Support for numerical logging (e.g. log the number of minutes of exercise per day rather than yes/no)
- Nice to have features
	- Shared habit tracking with friends
	- API access
	- Free
	- iOS widgets
	- Charts and visualizations in UI
