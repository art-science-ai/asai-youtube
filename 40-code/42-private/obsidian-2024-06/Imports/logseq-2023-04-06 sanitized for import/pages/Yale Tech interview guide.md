- Frameworks
	 - Product interview framework
		 - Clarify the problem
			 - What's the goal / objective?

		 - Structure / framework
			 - Summarize below points

		 - Users and customers

		 - Use cases
			 - Why are they using this product?

			 - What is the job to be done?

		 - Competitors and gaps
			 - How well is the competitor's product doing for these use cases?

			 - What are the gaps / weak spots?

		 - Features
			 - Features / changes / improvements

			 - Tie in to goals and use cases

		 - Wrap up
			 - Summarize and make a recommendation

	 - Go to market framework (also look at VCheng framework)
		 - Customers: total revenue, segments, purchase behavior, customer needs

		 - Competitors: total market size; market shares; market segments; direct vs. indirect competitors

		 - Company: strategy, culture, infrastructure, brand, etc.

		 - Collaborators: suppliers, distributors, etc.

		 - Climate: regulatory / economic / government / culture / etc.

	 - Product metrics framework
		 - User acquisition
			 - Number of users and growth

			 - Where are the users coming from?

		 - Activity
			 - Number of active users and growth

		 - Conversion and retention
			 - Net promoter score

			 - Churn rate

		 - Money
			 - Customer acquisition cost

			 - Customer lifetime value

			 - Profit per customer

- Questions to prepare in advance
	 - Tell me about 3 products you used in the past week?
		 - 1Password
			 - Problem: password security and convenience

			 - Users: all internet users; segment by security, convenience

			 - Use cases: create secure passwords; login; share

			 - Gaps: enterprise segment

			 - Improvements: on-prem,

		 - Airpods
			 - Problem: stress-free headset

			 - Users: iPhone users; segment by fitness, travel, everyday

			 - Use cases: phone calls, commuting, exercizing

			 - Gaps: noise isolation, fit

			 - Improvements: improve fit; release noise-isolating version

		 - Jupyter notebooks
			 - Problem: iPython / Jupyter notebooks implementation on cloud

			 - Users: students, academic researchers, data analysts

			 - Use cases: cross-device compatability, use from anywhere, cloud storage / env

			 - Gaps: collaboration, on-prem; speed

			 - Improvements: collaboration feature like google docs

		 - Overcast

		 - Apollo

		 - Insight timer

		 - Apple watch

		 - iPhone

		 - Google maps

		 - Asana

		 - Canvas

		 - Notes
			 - Talk about products for which you have an interesting vision / strategy
