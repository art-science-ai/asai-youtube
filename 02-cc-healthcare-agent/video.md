# Building AI Solutions with Claude Agent SDK

## Intro - Why Agent SDK

- Package complexity into simple interfaces
- Same power as Claude Code under the hood
- Subagents, tools, skills - all available programmatically
- First time using Claude Agent SDK - Learning it live by building a real product

## The problem

- Healthcare staff need to extract records from transcripts
- They need to generate reports
- They don't have time to learn developer tools

## The build

- Using Agent SDK to create specialized agents
- Web chat interface anyone can use
- Database integration for persistence

## Demo

- End user experience walkthrough
- Upload transcript → review extraction → generate PDF
- No terminal, no prompts, just results

## Wrap up

- What we learned about Agent SDK
- What worked, what was surprising
- The opportunity for developers
