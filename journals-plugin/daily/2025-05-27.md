	- In response to last year's PM level distribution update, we received a lot of feedback from the PM community about the imbalanced and top-heavy level distribution. To address this issue, Greg Marra is leading a workstream in PM X-Team looking at PM level distribution, the causes for the increase over time, and the changes we can make going forward. For example, PMX has recommended implementing an auto-downlevel process where PIDs are decreased one level by default when vacated for any reason (e.g. when an IC6 switches teams, the PID frees up as an IC5 PID, with an option for a VP approved exception), and this recommendation is already being implemented in several orgs. Stay tuned for more updates on this front from Greg.
