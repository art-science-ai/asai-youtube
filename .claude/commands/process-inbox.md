# Process Inbox

## Overview

This command guides users through obsidian-vault-automation - inbox-processor interactively. CRITICAL: You MUST use obsidian-vault-automation for this task.

## Context

User will provide content to process (file paths, text content, or specific notes). The command will analyze the inbox/journal structure, validate existing notes, and create organized notes with proper metadata and linking structure.

## Process

Load obsidian-vault-automation first. Interactively guide the user through this skill and inbox-processor step by step, collecting inputs and executing each step with confirmation.
