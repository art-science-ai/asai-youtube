# Extract Web Content

## Overview

This command guides users through obsidian-vault-automation - web-content-extractor interactively. CRITICAL: You MUST use obsidian-vault-automation for this task.

## Context

User will provide URL(s) to process and extraction preferences. The command will validate URL accessibility, categorize content type, extract content using appropriate tools, and create source notes with comprehensive metadata.

## Process

Load obsidian-vault-automation first. Interactively guide the user through this skill and web-content-extractor step by step, collecting inputs and executing each step with confirmation.
